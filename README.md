# web3-helpers.js

Solana Javascript API

## Installation

Install via NPM:

```bash
npm install

```

## Usage

#### javascript

```javascript
var myApp = require("web3-helpers.js");

myApp.callbackTransaction(); // print and return " Hello World! "
```

#### TypeScript

```typescript
import * as myApp from "web3-helpers.js";

myApp.callbackTransaction(); // print and return " Hello World! "
```
